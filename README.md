# ARMA3Mods

### SuperPuma BPol
  
  SuperPuma Helicopter for the German Federal Police.
  ACE3 required for fast roping.
  
  #### Requirements
  * [Spanish Army Mod FFAA](http://www.armaholic.com/page.php?id=25573)
  * [ACE3](https://ace3mod.com/)

  * src
    * source files
  * compiled

  * keys
